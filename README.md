# Programowanie-GUI-2026
Programowanie GUI - Portfolio Projektów (2026) 🚀
Repozytorium zawiera zestaw projektów akademickich realizowanych w ramach przedmiotu Programowanie GUI. Projekty skupiają się na wykorzystaniu języka Python, biblioteki Tkinter oraz zaawansowanych mechanikach symulacji i emulacji.

📂 Zawartość Repozytorium
🎮 Projekt 1: Arkanoid Retro
Klasyczna gra zręcznościowa zaimplementowana w środowisku graficznym.

Kluczowe funkcje: Silnik kolizji AABB, obsługa zdarzeń myszy, system niszczenia bloków.

Technologie: Python, Tkinter (Canvas).

🏭 Projekt 3: Symulator Linii Produkcyjnej (SCADA)
Zaawansowana aplikacja monitorująca parametry procesu przemysłowego w oparciu o realne dane systemowe.

Kluczowe funkcje: Mapowanie obciążenia CPU na temperaturę silników, dziennik zdarzeń (logi), system alarmowy.

Bezpieczeństwo: Implementacja mechanizmu Dead Man's Switch (autodiagnostyka operatora z 30-sekundowym czasem reakcji).

Biblioteki: psutil, random.

💻 Projekt 5: Emulator Mikroprocesora 16-bit
Symulator niskopoziomowej architektury procesora wzorowanej na modelu Intel 8086.

Model: 4 rejestry ogólnego przeznaczenia (AX, BX, CX, DX) z obsługą części 8-bitowych (H/L).

Instrukcje: MOV, ADD, SUB.

Tryby pracy: Wykonanie całościowe oraz tryb krokowy (Step-by-step) do analizy przepływu danych.

🛠 Narzędzia i Technologie
Język: Python 3.x

GUI: Tkinter

Wersjonowanie: Git / GitHub

Dokumentacja: LaTeX (Overleaf) – pełne sprawozdania techniczne dla każdego modułu.

🚀 Jak uruchomić projekty?
Sklonuj repozytorium:

Bash

git clone https://github.com/Kszyszkovski/Programowanie-GUI-2026.git
Zainstaluj wymagane biblioteki:

Bash

pip install psutil
Uruchom wybrany projekt:

Bash

python Projekt-3-Symulator/main.py


Jak to dodać na GitHub?
